# 🤖 ProDev AI — Full Stack Coding Assistant

ProDev AI is an **AI-powered coding assistant** built with **React**, **TailwindCSS**, and **Google Gemini API**.  
It delivers **complete, runnable code** and professional explanations — with a slick UI, syntax highlighting, typewriter animation, and instant copy-to-clipboard for code blocks.

---

## 🚀 Features

- 💬 **Real-time chat** with Google Gemini 1.5 Flash.
- ⌨️ **Typewriter effect** for AI responses.
- 🖌 **Markdown & syntax highlighting** with `react-markdown` and `rehype-highlight`.
- 📋 **One-click code copy** from AI output.
- 🎨 **Modern responsive UI** with TailwindCSS gradients & animations.
- 🔄 **Auto-scroll** to latest message.
- ⚡ **Professional developer system prompt** to ensure high-quality, runnable outputs.

---

## 🛠 Tech Stack

- **Frontend:** React + Vite
- **Styling:** TailwindCSS
- **Markdown Rendering:** react-markdown + remark-gfm
- **Syntax Highlighting:** rehype-highlight + highlight.js
- **AI Backend:** Google Gemini API (via Axios)
- **State Management:** React Hooks
